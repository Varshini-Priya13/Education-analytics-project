📊 Education Analytics Dashboard (OULAD + Synthetic Data)

🔍 Project Overview

This project focuses on analyzing student performance, engagement, and retention patterns using the Open University Learning Analytics Dataset (OULAD) combined with synthetic datasets to simulate real-world academic scenarios.

The goal is to derive actionable insights that can help educational institutions improve student success rates, identify at-risk learners, and optimize course delivery.

🎯 Objectives

Analyze student demographics and academic performance

Identify factors influencing student success and dropout rates

Track student engagement using LMS activity data

Build interactive dashboards for data-driven decision making

📂 Dataset Details

📘 OULAD Dataset

Contains data about students, courses, assessments, and interactions

Includes:

Student demographics

Assessment scores

Virtual Learning Environment (VLE) interactions



🧪 Synthetic Dataset

Created to simulate:

Attendance trends

Faculty Workload and Performance 

Alumni Contribution and analysis

Helps enhance real-world applicability

🛠️ Tools & Technologies

Power BI – Dashboard creation & visualizatio

DAX – Measures and KPIs

📊 Key Analysis Performed

📌 Student Performance Analysis

Pass/Fail distribution

Average scores across courses

Assessment-wise performance trends


📌 Engagement Analysis

VLE activity vs performance correlation

Active vs inactive students segmentation


📌 Dropout Prediction Indicators

Identification of at-risk students

Impact of engagement and attendance on dropout


📌 Demographic Insights

Age group vs performance

📈 Dashboard Features

Interactive filters (course, semester, demographics)

KPI cards:

Pass Rate

🔑 Key Insights

Higher VLE interaction strongly correlates with better performance

Students with low early engagement are more likely to drop out

Certain demographics show consistent performance gaps

Attendance + LMS activity together improve prediction accuracy
